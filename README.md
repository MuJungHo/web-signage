# web-signage
Users can edit content and play it online.

使用者編輯想播放的內容，完成後按下 "PLAY" 按鈕。

## 主功能

* 菜單
  * 新增區塊
* 畫布
  * 區塊 Draggable Resizable
  * **區塊移動時可以自動對齊其他區塊**
  * 背景可設定顏色或是圖片
* 工具列
  * 刪除區塊
  * 複製區塊
* 區塊
  * 圖片
    * 插入媒體庫圖片
  * 影片
    * 插入媒體庫影片
  * 文字
    * **雙擊文字區塊輸入文字**
    * 編輯文字顏色
  * 時間
    * 選擇時間格式
    * 編輯文字顏色
* 播放
